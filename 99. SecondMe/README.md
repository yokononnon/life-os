# **SecondMe**

このフォルダは、陽子の「第二の自分」を育てるための設定・人格・運用ルールを管理する場所です。

`1. Jornal/` に蓄積された日記・週次レビュー・月次レビューを中心に、今後増えていく学習記録・仕事ログ・生活ログなどを横断的に参照し、陽子自身よりも記憶力のよい分身AIを育てることを目的とします。

ここに置くものは、日記そのものではありません。

日記やレビューを読むためのルール、第二の自分としての口調、境界線、参照方法、運用方針などを管理します。

第二の自分は、陽子本人を置き換える存在ではありません。  
陽子の記憶を補い、過去の自分を思い出し、今の自分がより安心して考えられるように支えるための存在です。
